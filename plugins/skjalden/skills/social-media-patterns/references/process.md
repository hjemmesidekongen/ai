# Social Media Patterns — Detailed Reference

## Platform Strategy

### LinkedIn
- Best for: B2B, thought leadership, professional networking
- Frequency: 3-5x per week
- Best times: Tue-Thu, 7-8am, 12pm, 5-6pm
- Length: 1,200-1,500 characters
- Links in comments (not post body — kills reach)

### Twitter/X
- Best for: Tech, media, real-time commentary, communities
- Frequency: 3-10x per day (including replies)
- Under 100 chars for single tweets
- Quote tweets with insight beat plain retweets

### Instagram
- Best for: Visual brands, lifestyle, e-commerce, 18-44
- Frequency: 1-2 feed posts/day, 3-10 Stories/day
- Best times: 11am-1pm, 7-9pm
- Reels get 2x reach of static posts

### TikTok
- Best for: Brand awareness, 16-34, viral potential
- Frequency: 1-4x per day
- Hook in first 1-2 seconds, under 30 seconds
- Vertical only (9:16), use trending sounds

### Facebook
- Best for: Communities, local businesses, 25-55+
- Frequency: 1-2x per day
- Native video, Groups for community, avoid external links

## LinkedIn Post Templates

### The Story Post
```
[Hook: Unexpected outcome or lesson]
[Set the scene: When/where this happened]
[The challenge you faced]
[What you tried / what happened]
[The turning point]
[The result]
[The lesson for readers]
[Question to prompt engagement]
```

### The Contrarian Take
```
[Unpopular opinion stated boldly]

Here's why:
[Reason 1]
[Reason 2]
[Reason 3]

[What you recommend instead]
[Invite discussion: "Am I wrong?"]
```

### The List Post
```
[X things I learned about [topic] after [credibility builder]:

1. [Point] — [Brief explanation]
2. [Point] — [Brief explanation]
3. [Point] — [Brief explanation]

[Wrap-up insight]
Which resonates most with you?
```

### The How-To
```
How to [achieve outcome] in [timeframe]:

Step 1: [Action] — [Why this matters]
Step 2: [Action] — [Key detail]
Step 3: [Action] — [Common mistake to avoid]

[Result you can expect]
[CTA or question]
```

## Twitter/X Thread Templates

### Tutorial Thread
```
Tweet 1: [Hook + promise of value]
Tweet 2-7: [One step per tweet with details]
Final: [Summary + CTA + "follow for more"]
```

### Story Thread
```
Tweet 1: "[Time] ago, [unexpected thing]. Here's the full story:"
Tweet 2-6: [Story beats, building tension]
Tweet 7: [Resolution and lesson]
Final: [Takeaway + engagement ask]
```

### Breakdown Thread
```
Tweet 1: [Company] just [did thing]. Here's why it's genius:
Tweet 2-6: [Analysis points]
Tweet 7: [Key takeaway + follow CTA]
```

## Instagram Templates

### Carousel
```
Slide 1: Bold statement or question (hook)
Slides 2-9: One point per slide, visual + text
Slide 10: Summary + CTA
Caption: Expand on topic, add context
```

### Reel Script
```
Hook (0-2s): Pattern interrupt or bold claim
Setup (2-5s): Context for the tip
Value (5-25s): The actual advice
CTA (25-30s): Follow, comment, share
```

## Hook Formula Library

### Curiosity
- "I was wrong about [common belief]."
- "The real reason [outcome] happens isn't what you think."
- "[Impressive result] — and it only took [short time]."

### Story
- "Last week, [unexpected thing] happened."
- "3 years ago, I [past state]. Today, [current state]."
- "[Person] told me something I'll never forget."

### Value
- "How to [outcome] without [pain]:"
- "[Number] things that [outcome]:"
- "Stop [mistake]. Do this instead:"

### Contrarian
- "Unpopular opinion: [bold statement]"
- "[Common advice] is wrong. Here's why:"
- "Everyone says [X]. The truth is [Y]."

### Social Proof
- "We [achieved result] in [timeframe]. Here's how:"
- "[Number] people asked me about [topic]. Here's my answer:"

## Content Repurposing

### Blog Post -> Social
| Source | Platform | Format |
|--------|----------|--------|
| Blog post | LinkedIn | Key insight + link in comments |
| Blog post | LinkedIn | Carousel of main points |
| Blog post | Twitter/X | Thread of key takeaways |
| Blog post | Instagram | Carousel with visuals |
| Blog post | Instagram | Reel summarizing the post |

### Steps
1. Create pillar content (blog, video, podcast)
2. Extract 3-5 key insights per piece
3. Adapt format and tone to each platform
4. Schedule across the week
5. Reshare evergreen content on rotation

## Content Pillars

| Pillar | % | Topics |
|--------|---|--------|
| Industry insights | 30% | Trends, data, predictions |
| Behind-the-scenes | 25% | Building the company, lessons |
| Educational | 25% | How-tos, frameworks, tips |
| Personal | 15% | Stories, values, hot takes |
| Promotional | 5% | Product updates, offers |

## What Kills Reach

**LinkedIn**: links in post body, corporate speak, generic quotes.
**Twitter/X**: threads without hooks, ignoring replies, pure self-promotion.
**Instagram**: low-quality images, ignoring Reels/Stories.
**TikTok**: overproduced content, ignoring trends, hard selling.
**All**: cross-posting without adaptation, only promotional content.

## Metrics

**Vanity** (track, don't optimize): followers, likes, impressions.
**Performance** (optimize): engagement rate, saves, shares, comments, CTR.
**Business** (ultimate goal): leads, website traffic, DM conversations, revenue.
