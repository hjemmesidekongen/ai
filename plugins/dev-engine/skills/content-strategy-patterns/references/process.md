# Content Strategy Patterns — Detailed Reference

## Content Pillar Strategy

### Ratio Guidelines
```
Educational:    40%  — How-tos, tutorials, best practices, tips
Inspirational:  25%  — Success stories, case studies, transformations, vision
Conversational: 25%  — Behind-the-scenes, team spotlights, Q&As, polls
Promotional:    10%  — Product updates, offers, event announcements, CTAs
```

Track monthly. If promotional exceeds 15%, audience trust erodes. If educational drops below 30%, perceived expertise weakens.

### Monthly Calendar Structure
```
Week 1:
  Monday:    Educational (blog post)
  Wednesday: Inspirational (social media)
  Friday:    Conversational (email/community)

Week 2:
  Monday:    Educational (video/guide)
  Wednesday: Case study
  Friday:    Curated content

Week 3:
  Monday:    Educational (infographic)
  Wednesday: Behind-the-scenes
  Friday:    Community spotlight

Week 4:
  Monday:    Monthly roundup
  Wednesday: Thought leadership
  Friday:    Promotional
```

## Blog Post Templates

### How-To Guide
```
# How to [Achieve Desired Outcome] in [Timeframe]

## Introduction
- Hook: question or surprising fact
- Problem statement
- What reader will learn
- Why it matters now

## Prerequisites / What You'll Need
- Tool/Resource list
- Estimated time

## Step 1: [Action]
- Clear instruction
- Why this step matters
- Common mistakes to avoid
- Visual aid or example

## Step 2-N: [Action]
(Repeat structure)

## Troubleshooting Common Issues
### Issue 1: [Problem]
**Solution**: [Fix]

## Results You Can Expect
- Immediate outcomes
- Long-term benefits
- Success metrics

## Next Steps
- Advanced techniques
- Related guides
- CTA
```

### Listicle
```
# [Number] [Adjective] Ways to [Achieve Goal]

## Introduction
- Context/trend driving this topic
- Promise of what reader gains

## 1. [Most Important Item]
**Why it matters**: [Brief explanation]
**How to implement**: [2-3 actionable steps]
**Pro tip**: [Expert insight]
**Example**: [Real-world application]

## 2-N. [Items]
(Repeat structure)

## Bringing It All Together
- How items work together
- Priority order
- Expected timeline

## Your Action Plan
1. Start with [easiest]
2. Progress to [next]
3. Measure [metrics]
```

### Case Study
```
# How [Company] Achieved [Result] Using [Solution]

## Executive Summary
- Company overview, challenge, solution, key results (3 metrics)

## The Challenge
- Industry context, company situation, previous attempts
- Specific pain points (3)

## The Solution
- Discovery process, strategic approach, why this solution
- Implementation phases with timelines

## The Results
- Quantitative: X% increase, $Y saved, Z improvement
- Qualitative: team feedback, customer response

## Key Takeaways
1. Lesson learned
2. Best practice discovered
3. Unexpected benefit

## CTA: How You Can Achieve Similar Results
```

### Thought Leadership
```
# [Provocative Statement About Industry Future]

## The Current State
- Industry snapshot, prevailing wisdom, why status quo fails

## The Emerging Trend
- 3 drivers (technology/market/behavior)
- Evidence and examples

## Implications
- Short-term (6-12 months): adjustments, quick wins, risks
- Long-term (2-5 years): fundamental shifts, opportunities

## Strategic Recommendations
- For leaders: priorities, investment areas
- For practitioners: skill development, tools

## The Path Forward
- Call to action, next steps
```

## Social Media Templates

### LinkedIn Post
```
Hook/Pattern Interrupt (first line is all that shows)

Context paragraph — situation or challenge.

Key insight or lesson:
- Bullet 1 (specific detail)
- Bullet 2 (measurable outcome)
- Bullet 3 (unexpected discovery)

Brief story illustrating the point.

Takeaway with clear value.

Question to encourage engagement?

#Hashtag1 #Hashtag2 #Hashtag3
```

### Twitter/X Thread
```
1/ Bold opening statement or question
2/ Context — why this matters now
3/ Problem most people face
4/ Conventional solution (and why it falls short)
5/ Better approach — introduction
6-9/ Steps of better approach (action + why it works)
10/ Real example or case study
11/ Common objection addressed
12/ Recap in 3 key points
13/ CTA
14/ P.S. — bonus insight
```

```

[Key insight]

3 things that changed everything:
1. [First point]
2. [Second point]
3. [Third point]

[Question to audience]

#hashtag1 #hashtag2 ... (10-30 relevant)
```

## Email Templates

### Newsletter
```
Subject: [Benefit] + [Urgency/Curiosity]
Preview: [Complements subject, doesn't repeat]

Hi [Name],

[Personal observation or timely hook — 1-2 sentences]
[Transition to main topic — why reading this matters]

## Main Content
- Point 1: benefit-focused
- Point 2: specific example
- Point 3: actionable tip

[Brief elaboration on most important point]

## Resource of the Week
[Title with link + one sentence on value]

## Quick Win You Can Implement Today
[Specific tip — 2-3 steps max]

[Closing thought]
[Signature]

P.S. [Additional value or soft CTA]
```

### Promotional Email
```
Subject: [Specific benefit] by [deadline]
Preview: [Scarcity or exclusivity element]

Hi [Name],

[Acknowledge pain point]
[Agitate — why this problem persists]

I've got something that can help:
[Solution introduction]

Here's what you get:
- Benefit 1 (not feature)
- Benefit 2 (not feature)
- Benefit 3 (not feature)

[Social proof — testimonial or results]
[Handle main objection]

[CTA button]

[Urgency element]

P.S. [Reinforce urgency or add bonus]
```

## Video Script Template
```

Section 2: [Key Point] — repeat structure
Section 3: [Key Point] — repeat structure

[Final 15-30s]

```

## Content Repurposing Matrix

```
Original: Blog Post (2000 words)
├── Social Media
│   ├── 5 Twitter posts (key quotes)
│   ├── 1 LinkedIn article (executive summary)
│   ├── 3 Instagram carousels (main points)
│   └── 1 Facebook post (intro + link)
├── Email
│   └── Newsletter feature (summary + CTA)
├── Video
│   ├── YouTube explainer (script from post)
│   └── Short-form video (quick tips)
├── Audio
│   └── Podcast talking points
└── Visual
    ├── Infographic (data points)
    └── Slide deck (presentation)
```

## SEO Content Structure

```
URL: /primary-keyword-secondary-keyword

Title Tag: Primary Keyword - Secondary Benefit | Brand (under 60 chars)
Meta Description: Action verb + primary keyword + benefit + CTA (155 chars)

# H1: Primary Keyword + Unique Angle

Introduction (50-100 words)
- Primary keyword in first 100 words
- State what reader will learn

## H2: Secondary Keyword Variation 1
[Content with LSI keywords naturally integrated]

### H3: Specific subtopic
- Detail points

## H2: Related Questions (FAQ Schema)
### Question 1?
[Concise answer with keyword]

## Conclusion
- Recap, include primary keyword, clear next action

Internal Links: 2-3 relevant articles
External Links: 1-2 authoritative sources
```

## Quality Checklists

### Pre-Publishing
- [ ] Keyword research completed
- [ ] Title under 60 characters
- [ ] Meta description written (155 chars)
- [ ] Headers properly structured (H1, H2, H3)
- [ ] Internal links added (2-3)
- [ ] Images optimized with alt text
- [ ] CTA included and clear
- [ ] Proofread and fact-checked
- [ ] Mobile preview checked

### Content Quality
- [ ] Addresses specific audience need
- [ ] Provides unique value/perspective
- [ ] Includes actionable takeaways
- [ ] Uses appropriate brand voice
- [ ] Contains supporting data/examples
- [ ] Free of jargon
- [ ] Scannable format (bullets, headers)
- [ ] Engaging hook in introduction
- [ ] Clear conclusion and next steps
